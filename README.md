## 🧑‍💻 Getting Started

To run this project locally:

### 1. Clone the repo

```bash
git clone https://github.com/RADHIKA281005/23BKT0078_L37_MJ_radhika.git
cd 23BKT0078_L37_MJ_radhika/react-website-yt
```

### 2. Install dependencies

```bash
npm install
```

### 3. Run the development server

```bash
npm start
```
